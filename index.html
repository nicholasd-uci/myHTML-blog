What is Bootstrap?


Simply Put... This is Bootstrap!

This is a Div box inside of a Div Row tag. These tag's are brought in by using bootstrap CDNs and their Script tags'. The column is a small-12 which is a built in feature from bootstrap letting the column be classified as such. Learn More Here. As a Jr. Web Developer I would highly recommend that you read over the Grid as well as the Utilities for layout, sections as well. It will help you get a better grasp on how to layout and envision your website and what you want it to be. Then once you have done that, go ahead and move on to Content Section and start coding away.

Happy Coding! 👾

In a further explanation about the page, this Row sits inside of our Container Tag, that is a separate Div within itself. This Row with information along with other Divs' in this particular code are kept inside of this Container letting HTML know to render the information inside of it. You can then choose from a wide range of schemes and even choose other Front-End Frameworks to better suit what you want to get done.

Think of it like a picture frame. You place certain photos were you want them to go and they can only fit inside of that frame, prior to sealing it with glass. So you make sure to align all of your photo's, you might have to cut some of them, overlap them, and even bring in some type of design to suit the photo collage as the background.
